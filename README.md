A script that fetches ASN information and ip ranges from [Hurricane Electric](https://bgp.he.net.).

### Usage

**Single target**
```
python heASNs.py -i dell
```

**Multiple targets**
```
python heASNs.py -i /path/to/org/names/file.txt
```
